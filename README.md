# 🎯 SRT 표 사냥

원하는 SRT 표가 매진이어도 **빈자리가 나면 자동으로 잡아 텔레그램으로 알려주는** 웹앱이에요.
폰에서 조건만 걸어두면, 컴퓨터(서버)가 계속 빈자리를 조회해서 예매합니다.

- **프론트엔드**: `webapp_mockup.html` — 출발/도착·날짜·시간대·좌석·인원·자동결제 UI
- **백엔드**: `server.py` (FastAPI) — 조건을 받아 백그라운드로 예매 루프 실행
- **예매 라이브러리**: `srtgo/` (SRT API)

## 빠른 시작
```bash
pip install -r requirements.txt
python server.py
# 브라우저에서 http://localhost:8000 접속 → 처음 설정에서 계정·텔레그램 입력
```

자세한 설치·배포·폰 접속(Tailscale) 방법은 [설치가이드.md](설치가이드.md) 참고.

> ⚠️ 계정·카드·텔레그램 정보는 실행하는 컴퓨터에만 저장되며 저장소에 포함되지 않습니다.
> KTX(코레일)는 매크로 차단으로 동작하지 않고, SRT만 지원합니다.
