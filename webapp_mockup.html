표가 잡히면 여기로 알림이 와요. 순서대로 하면 돼요:
  1. 텔레그램에서 @BotFather 검색 → 대화 시작
  2. /newbot 입력 → 봇 이름 정하기 → 나오는 토큰 복사
  3. @userinfobot 검색 → 아무 메시지 → 나오는 Id 숫자가 채팅 ID
