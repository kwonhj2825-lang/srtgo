# 🎯 SRT 표 사냥꾼 — 설치 가이드

원하는 SRT 표가 매진이어도, **빈자리가 나면 대신 잡아서 텔레그램으로 알려주는** 프로그램이에요.
폰에서 조건만 걸어두면, 컴퓨터가 계속 표를 사냥합니다.

> ⚠️ **각자 자기 컴퓨터에 설치해서 씁니다.** (내 SRT 계정·텔레그램으로 동작하고, 내 컴퓨터가 표를 잡아요.)
> 그래서 이 폴더를 통째로 받아서 아래 순서대로 하면 됩니다.

---

## 준비물
- **켜둘 수 있는 Windows 컴퓨터** (표 사냥 중엔 켜져 있어야 해요)
- **SRT 계정** (평소 예매하는 그 계정)
- **텔레그램** (알림 받을 앱)
- (선택) **폰** — 밖에서도 쓰려면

---

## 1. 파이썬 설치 (이미 있으면 건너뛰기)
1. https://www.python.org/downloads/ 에서 최신 버전 설치
2. 설치 첫 화면에서 **"Add Python to PATH"** 체크 ✅ 꼭!

## 2. 프로그램 폴더 준비
- 받은 `srtgo` 폴더를 원하는 위치에 두기 (예: `C:\srtgo`)
- 그 안에 `server.py`, `webapp_mockup.html`, `requirements.txt`, `srtgo\` 폴더가 있어야 해요

## 3. 필요한 것 설치
PowerShell(또는 명령 프롬프트)을 열고, 폴더로 이동해서 한 줄 실행:

```powershell
cd C:\srtgo
pip install -r requirements.txt
```

## 4. 서버 실행
```powershell
python server.py
```
- `Uvicorn running on http://0.0.0.0:8000` 이 뜨면 성공 🎉
- **이 창을 켜둔 채로** 다음으로 넘어가세요

## 5. 첫 설정 (브라우저에서)
1. 브라우저에서 **http://localhost:8000** 접속
2. 처음엔 **"처음 설정 🎯"** 화면이 자동으로 떠요
3. 안내대로 입력:
   - **SRT 계정** (아이디·비밀번호)
   - **텔레그램 연결**
     - 텔레그램에서 **@BotFather** → `/newbot` → 봇 만들고 **토큰** 복사
     - **@userinfobot** 에게 아무 메시지 → 나오는 **Id 숫자**가 채팅 ID
4. **저장하고 시작하기** → 끝! 이제 조건 걸고 **표 사냥 시작** 하면 돼요

> 나중에 계정을 바꾸려면 화면 오른쪽 위 **⚙️ 설정**에서 다시 열 수 있어요.

---

## 📱 (선택) 폰에서 어디서든 쓰기 — Tailscale
집 밖에서도 폰으로 쓰고 싶으면:

1. 컴퓨터에 **Tailscale** 설치 (https://tailscale.com/download) → 로그인
2. 폰에도 **Tailscale 앱** 설치 → **같은 계정**으로 로그인 → 연결 ON
3. 컴퓨터에서 Tailscale IP 확인: PowerShell에 `tailscale ip -4` (예: `100.x.x.x`)
4. 폰 브라우저에서 **http://<그 IP>:8000** 접속
   - 폰 Tailscale이 켜져 있으면 밖에서도(LTE/5G) 접속돼요
   - 표 사냥은 컴퓨터가 하니까, 폰 Tailscale을 꺼도 알림은 옵니다

---

## 🔌 (선택) 컴퓨터 켜면 자동 실행 + 절전 끄기
매번 `python server.py` 치기 귀찮으면:

**자동 실행** — `run_hidden.vbs` 파일을 아래 폴더에 복사(경로는 본인 것으로 수정):
```
C:\Users\<사용자>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```
(`start_server.bat` 안의 python 경로와 폴더 경로도 본인 환경에 맞게 수정하세요.)

**절전 끄기** — PowerShell에서:
```powershell
powercfg /change standby-timeout-ac 0
powercfg /change hibernate-timeout-ac 0
```

## ⚠️ 알아두기
- **컴퓨터가 켜져 있어야** 표 사냥이 됩니다 (꺼지면 멈춰요).
- **[표 사냥 시작]을 누르면 진짜로 표가 예매**되고, 자동 결제를 켜면 결제까지 됩니다. 조건을 잘 확인하세요.
- 계정·카드 정보는 **내 컴퓨터에만** 저장됩니다 (외부로 안 나가요).
- SRT만 지원해요 (KTX/코레일은 매크로 차단으로 동작하지 않습니다).
